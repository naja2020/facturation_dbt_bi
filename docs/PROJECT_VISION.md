# Facturation DBT BI

## Vision

Développer une plateforme de gestion financière personnelle inspirée des applications professionnelles.

L'application permettra :

- gérer plusieurs utilisateurs
- gérer plusieurs comptes bancaires
- enregistrer toutes les transactions
- gérer plusieurs devises
- gérer des objectifs d'épargne
- générer automatiquement des factures
- produire des tableaux de bord Power BI
- transformer les données avec dbt
- historiser les données
- appliquer des contrôles qualité
- versionner le projet avec GitHub
